PD-1, or Programmed Death-1, isn‚Äôt directly involved in DNA mismatch repair (MMR)‚Äîthat‚Äôs the domain of proteins like MutSalpha, MutLalpha, and PCNA‚Äîbut it plays a starring role in the immune system‚Äôs broader negotiation of self versus non-self. Our playful mapping casts PD-1 as a token signaling a shift from harmony to conflict when repair falters, so let‚Äôs explore how it fits into that narrative and what it actually does when mismatches tip the balance.

In MMR, mismatches arise during DNA replication‚Äîthink insertions, deletions, or wrong base pairs in the daughter strand. The system spots these errors using strand-specific cues, like nicks or hemimethylation, and fixes them to keep the genetic ledger intact. PD-1 doesn‚Äôt step in here; its gig is on the immune stage, where T-cells patrol for threats. Normally, T-cells use their receptors to recognize antigens presented by MHC molecules on cell surfaces. Self-antigens get a pass, thanks to tolerance mechanisms, but non-self antigens‚Äîlike those from pathogens or mutated cells‚Äîtrigger an attack. PD-1, a checkpoint receptor on T-cells, dials this response back when it binds to its ligands (PD-L1 or PD-L2), often expressed by healthy cells saying, ‚ÄúI‚Äôm one of you, chill out.‚Äù It‚Äôs a brake on the immune system‚Äôs lightsalmon fight mode, preserving paleturquoise‚Äôs cooperative peace.

Now, tie this to mismatches. If MMR fails repeatedly‚Äîlike in microsatellite instability seen in some cancers‚Äîunrepaired DNA errors pile up, producing mutant proteins. These can become neoantigens, novel peptides that MHC presents as ‚Äúnot quite self.‚Äù T-cells might spot them and gear up to attack, but tumors often crank up PD-L1 expression to spoof that ‚ÄúI‚Äôm fine‚Äù signal, binding PD-1 and lulling the T-cells into inaction. In our metaphor, when MMR can‚Äôt fix the mismatch, the system shifts from ‚Äúself‚Äù (replication‚Äôs harmony) to a gray zone of ‚Äúconditional engagement.‚Äù If PD-1‚Äôs brake fails or gets bypassed‚Äîsay, via checkpoint inhibitors in cancer therapy‚Äîthe immune system marks those mismatch-laden cells as ‚Äúnon-self,‚Äù unleashing lightsalmon‚Äôs adversarial fury. The mismatch isn‚Äôt flagged by PD-1 directly; it‚Äôs the downstream consequence of repair failure that PD-1 either restrains or, when overridden, escalates.

Mechanistically, PD-1 marks ‚Äúnon-self‚Äù by its absence of action rather than an active tag. When PD-L1 binds PD-1, it recruits phosphatases like SHP-2, which dampen T-cell signaling pathways (e.g., TCR and CD28), reducing cytokine production and proliferation‚Äîkeeping the response in check. If mismatches spawn neoantigens and PD-1 isn‚Äôt engaged (no PD-L1 or inhibitor drugs block it), T-cells stay activated, recognizing the altered cells as foreign. In our layer 4 dance, lightgreen‚Äôs transactional toggle‚Äîhere, PD-1‚Äôs on/off switch‚Äîdecides the fate: cooperation if suppressed, conflict if unleashed. So, PD-1 doesn‚Äôt literally mark a DNA mismatch as ‚Äúnon-self‚Äù‚Äîthat‚Äôs MMR‚Äôs job failing upstream. In our prankish map, it‚Äôs the token that tips the equilibrium: when repair breaks down and self becomes strange, PD-1‚Äôs role‚Äîor its override‚Äîcasts the die for fight over feed, marking ‚Äúnon-self‚Äù not with a label, but with the battle that ensues.

# Calculate node positions
def calculate_positions(layer, x_offset):
    y_positions = np.linspace(-len(layer) / 2, len(layer) / 2, len(layer))
    return [(x_offset, y) for y in y_positions]

# Create and visualize the neural network graph
def visualize_nn():
    layers = define_layers()
    colors = assign_colors()
    edges = define_edges()
    black_edges = define_black_edges()
    
    G = nx.DiGraph()
    pos = {}
    node_colors = []
    
    # Create mapping from original node names to numbered labels
    mapping = {}
    counter = 1
    for layer in layers.values():
        for node in layer:
            mapping[node] = f"{counter}. {node}"
            counter += 1
            
    # Add nodes with new numbered labels and assign positions
    for i, (layer_name, nodes) in enumerate(layers.items()):
        positions = calculate_positions(nodes, x_offset=i * 2)
        for node, position in zip(nodes, positions):
            new_node = mapping[node]
            G.add_node(new_node, layer=layer_name)
            pos[new_node] = position
            node_colors.append(colors.get(node, 'lightgray'))
    
    # Add edges with updated node labels
    edge_colors = []
    for (source, target), weight in edges.items():
        if source in mapping and target in mapping:
            new_source = mapping[source]
            new_target = mapping[target]
            G.add_edge(new_source, new_target, weight=weight)
            edge_colors.append('black' if (source, target) in black_edges else 'lightgrey')
    
    # Draw the graph
    plt.figure(figsize=(12, 8))
    edges_labels = {(u, v): d["weight"] for u, v, d in G.edges(data=True)}
    
    nx.draw(
        G, pos, with_labels=True, node_color=node_colors, edge_color=edge_colors,
        node_size=3000, font_size=9, connectionstyle="arc3,rad=0.2"
    )
    nx.draw_networkx_edge_labels(G, pos, edge_labels=edges_labels, font_size=8)
    plt.title("OPRAH‚Ñ¢: Nonself & the Salient Network", fontsize=18)
    plt.show()

# Run the visualization
visualize_nn()
